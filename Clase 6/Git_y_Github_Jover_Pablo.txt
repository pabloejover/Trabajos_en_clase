1-¿Quién inventó el sistema de control de versión Git y por qué?
Git es un software de control de versiones diseñado por Linus Torvalds. Tras probar muchas de las soluciones existentes y descubrir que ninguna cumplía con sus requisitos, Torvalds decidió diseñar e implementar su propio sistema de control de versiones.
